## 第一章 有理数

### 1.1 正数和负数

**正数**
- 大于0的数叫做正数
- 表示方法：+5、+3.2、+1/2（正号可省略）
- 实际意义：收入、上升、零上温度等

**负数**
- 在正数前面加上"-"号的数叫做负数
- 表示方法：-5、-3.2、-1/2
- 实际意义：支出、下降、零下温度等

**0的特殊性**
- 0既不是正数，也不是负数
- 0是正数和负数的分界点

**相反意义的量**
- 用正数和负数可以表示具有相反意义的量
- 例：收入100元记作+100，支出80元记作-80

### 1.2 有理数

**有理数的定义**
- 整数和分数统称为有理数
- 整数包括：正整数、0、负整数
- 分数包括：正分数、负分数

**有理数的分类**

```
有理数
├── 整数
│   ├── 正整数（1, 2, 3, ...）
│   ├── 零（0）
│   └── 负整数（-1, -2, -3, ...）
└── 分数
    ├── 正分数（1/2, 3.5, ...）
    └── 负分数（-1/2, -3.5, ...）
```

**数轴**
- 三要素：原点、正方向、单位长度
- 画法：
  1. 画一条水平直线
  2. 取一点表示0（原点）
  3. 规定向右为正方向
  4. 选取适当长度为单位长度

**数轴上的点与有理数的关系**
- 每一个有理数都可以用数轴上的一个点来表示
- 正数在原点右边，负数在原点左边
- 数轴上右边的数总比左边的数大

### 1.3 相反数

**定义**
- 只有符号不同的两个数互为相反数
- 0的相反数是0

**性质**
- 若a与b互为相反数，则a + b = 0
- 若a + b = 0，则a与b互为相反数
- 在数轴上，互为相反数的两个点位于原点两侧，且到原点距离相等

**求法**
- 在一个数前面添上"-"号，就得到它的相反数
- 例：5的相反数是-5，-3的相反数是3

### 1.4 绝对值

**定义**
- 数轴上表示数a的点与原点的距离叫做数a的绝对值
- 记作：|a|

**性质**
```
|a| = {
    a,    当 a > 0 时
    0,    当 a = 0 时
    -a,   当 a < 0 时
}
```

**重要结论**
- |a| ≥ 0（绝对值具有非负性）
- 若|a| = |b|，则a = b 或 a = -b
- 若|a| + |b| = 0，则a = 0且b = 0

**比较有理数大小**
- 正数 > 0 > 负数
- 两个正数，绝对值大的数大
- 两个负数，绝对值大的数反而小

### 1.5 有理数的加减法

**加法法则**
1. 同号两数相加：取相同符号，绝对值相加
   - (+3) + (+5) = +8
   - (-3) + (-5) = -8

2. 异号两数相加：取绝对值较大数的符号，用较大绝对值减去较小绝对值
   - (+7) + (-3) = +4
   - (-7) + (+3) = -4

3. 互为相反数相加得0
   - (+5) + (-5) = 0

4. 一个数同0相加，仍得这个数
   - (+3) + 0 = +3

**加法运算律**
- 交换律：a + b = b + a
- 结合律：(a + b) + c = a + (b + c)

**减法法则**
- 减去一个数，等于加上这个数的相反数
- a - b = a + (-b)

**加减混合运算**
- 先把减法统一成加法
- 省略加号和括号，写成省略括号的和的形式
- 运用运算律简化计算

### 1.6 有理数的乘除法

**乘法法则**
1. 两数相乘，同号得正，异号得负，绝对值相乘
   - (+3) × (+4) = +12
   - (-3) × (-4) = +12
   - (+3) × (-4) = -12

2. 任何数与0相乘，都得0

**乘法运算律**
- 交换律：a × b = b × a
- 结合律：(a × b) × c = a × (b × c)
- 分配律：a × (b + c) = a × b + a × c

**倒数**
- 乘积是1的两个数互为倒数
- 0没有倒数
- 正数的倒数是正数，负数的倒数是负数

**除法法则**
- 除以一个不等于0的数，等于乘这个数的倒数
- a ÷ b = a × (1/b) (b ≠ 0)
- 两数相除，同号得正，异号得负，绝对值相除
- 0除以任何一个不等于0的数，都得0

### 1.7 有理数的乘方

**定义**
- 求n个相同因数a的积的运算叫做乘方
- 记作：aⁿ
- a叫做底数，n叫做指数，结果叫做幂

**读法**
- aⁿ 读作"a的n次方"或"a的n次幂"
- 2³ 读作"2的3次方"或"2的3次幂"

**符号规律**
- 正数的任何次幂都是正数
- 负数的奇次幂是负数，偶次幂是正数
- 0的任何正整数次幂都是0

**重要结论**
- (-1)ⁿ = { -1, n为奇数; 1, n为偶数 }
- 1ⁿ = 1
- 0ⁿ = 0 (n为正整数)

### 1.8 科学记数法

**定义**
- 把一个大于10的数表示成 a × 10ⁿ 的形式
- 其中 1 ≤ a < 10，n是正整数

**表示方法**
- 5230000 = 5.23 × 10⁶
- 小数点向左移动6位，指数就是6

**近似数**
- 与实际数接近但不完全相等的数
- 精确度：精确到哪一位
- 有效数字：从左边第一个不是0的数字起，到精确到的位数止

## 第二章 整式的加减

### 2.1 单项式

**定义**
- 由数与字母的积组成的代数式叫做单项式
- 单独的一个数或一个字母也是单项式

**系数**
- 单项式中的数字因数叫做这个单项式的系数
- 例：-3x²y 的系数是 -3

**次数**
- 一个单项式中，所有字母的指数的和叫做这个单项式的次数
- 例：-3x²y 的次数是 3（2+1）

### 2.2 多项式

**定义**
- 几个单项式的和叫做多项式

**项**
- 多项式中的每个单项式叫做多项式的项
- 不含字母的项叫做常数项

**次数**
- 多项式里次数最高项的次数，叫做这个多项式的次数
- 例：3x² - 2x + 1 是二次三项式

**整式**
- 单项式和多项式统称整式

### 2.3 同类项

**定义**
- 所含字母相同，并且相同字母的指数也相同的项叫做同类项
- 几个常数项也是同类项

**合并同类项**
- 把多项式中的同类项合并成一项
- 法则：同类项的系数相加，所得结果作为系数，字母和字母的指数不变

### 2.4 去括号与添括号

**去括号法则**
- 括号前是"+"号，把括号和它前面的"+"号去掉后，原括号里各项的符号都不改变
  - +(a - b) = a - b
- 括号前是"-"号，把括号和它前面的"-"号去掉后，原括号里各项的符号都要改变
  - -(a - b) = -a + b

**添括号法则**
- 添括号后，括号前面是"+"号，括到括号里的各项都不变符号
- 添括号后，括号前面是"-"号，括到括号里的各项都改变符号

### 2.5 整式的加减

**运算法则**
- 几个整式相加减，通常用括号把每一个整式括起来，再用加减号连接
- 先去括号，再合并同类项

**化简求值**
- 先化简整式，再代入数值计算

## 第三章 一元一次方程

### 3.1 从算式到方程

**方程的定义**
- 含有未知数的等式叫做方程

**一元一次方程**
- 只含有一个未知数（元）
- 未知数的次数都是1
- 等号两边都是整式

**方程的解**
- 使方程左右两边相等的未知数的值
- 解方程：求方程的解的过程

**等式的性质**
- 性质1：等式两边加（或减）同一个数（或式子），结果仍相等
  - 若 a = b，则 a ± c = b ± c
- 性质2：等式两边乘同一个数，或除以同一个不为0的数，结果仍相等
  - 若 a = b，则 ac = bc
  - 若 a = b，c ≠ 0，则 a/c = b/c

### 3.2 解一元一次方程（一）—— 合并同类项与移项

**合并同类项**
- 把方程中含未知数的项合并
- 例：3x + 2x = 10 → 5x = 10

**移项**
- 把等式一边的某项变号后移到另一边
- 移项要变号！
- 例：3x + 5 = 2x - 7
  - 移项：3x - 2x = -7 - 5
  - 合并：x = -12

### 3.3 解一元一次方程（二）—— 去括号与去分母

**去括号**
- 按去括号法则先去括号
- 括号前是负号，括号内各项要变号

**去分母**
- 在方程两边同乘各分母的最小公倍数
- 不要漏乘不含分母的项
- 分子是多项式要加括号

**解方程步骤**
1. 去分母
2. 去括号
3. 移项
4. 合并同类项
5. 系数化为1

### 3.4 实际问题与一元一次方程

**基本步骤**
1. 审：审题，分析已知量和未知量
2. 设：设未知数（直接设或间接设）
3. 列：找等量关系，列方程
4. 解：解方程
5. 验：检验答案是否正确
6. 答：写出答案

**常见类型**
- 和差倍分问题
- 行程问题
- 工程问题
- 利润问题
- 配套问题
- 年龄问题

## 第四章 几何图形初步

### 4.1 几何图形

**立体图形**
- 各部分不都在同一平面内的图形
- 柱体：圆柱、棱柱
- 锥体：圆锥、棱锥
- 球体

**平面图形**
- 各部分都在同一平面内的图形
- 点、线、面、三角形、四边形、圆等

**三视图**
- 主视图（从正面看）
- 左视图（从左面看）
- 俯视图（从上面看）

**展开图**
- 将立体图形的表面适当剪开，可以展开成平面图形

### 4.2 直线、射线、线段

**直线**
- 经过两点有一条直线，并且只有一条直线（两点确定一条直线）
- 直线没有端点，向两方无限延伸

**射线**
- 直线上一点和它一旁的部分
- 射线有一个端点，向一方无限延伸

**线段**
- 直线上两点及两点间的部分
- 线段有两个端点，不能延伸

**线段的长短比较**
- 度量法：用刻度尺量出长度再比较
- 叠合法：将两条线段的一个端点重合，另一个端点放在同一侧比较

**线段的和差**
- AB + BC = AC（点B在线段AC上）

**线段的中点**
- 点M把线段AB分成相等的两条线段AM与BM
- AM = BM = 1/2 AB

**两点间的距离**
- 连接两点的线段的长度

### 4.3 角

**角的定义**
- 有公共端点的两条射线组成的图形
- 这个公共端点是角的顶点，这两条射线是角的两条边

**角的表示方法**
- 用三个大写字母：∠AOB（顶点写在中间）
- 用一个大写字母：∠O（顶点处只有一个角时）
- 用一个希腊字母：∠α、∠β
- 用一个数字：∠1、∠2

**角的度量**
- 度、分、秒
- 1° = 60′，1′ = 60″

**角的比较**
- 度量法：用量角器量出度数再比较
- 叠合法：将两个角的顶点及一边重合，另一边放在重合边的同侧比较

**角的和差**
- ∠AOC = ∠AOB + ∠BOC（OB在∠AOC内部）

**角的平分线**
- 从一个角的顶点出发，把这个角分成相等的两个角的射线
- ∠AOC = ∠BOC = 1/2 ∠AOB

**余角和补角**
- 余角：如果两个角的和等于90°，就说这两个角互为余角
  - ∠1 + ∠2 = 90°，∠1是∠2的余角，∠2也是∠1的余角
- 补角：如果两个角的和等于180°，就说这两个角互为补角
  - ∠1 + ∠2 = 180°，∠1是∠2的补角，∠2也是∠1的补角

**性质**
- 同角（等角）的余角相等
- 同角（等角）的补角相等

**方位角**
- 表示方向的角
- 以正北、正南方向为基准

## 第五章 相交线与平行线

### 5.1 相交线

**邻补角**
- 两个角有一条公共边，它们的另一边互为反向延长线
- 性质：邻补角互补（和为180°）

**对顶角**
- 两个角有一个公共顶点，并且一个角的两边分别是另一个角两边的反向延长线
- 性质：对顶角相等

**垂线**
- 当两条直线相交所成的四个角中，有一个角是直角时，就说这两条直线互相垂直
- 记作：a⊥b
- 交点叫垂足

**垂线的性质**
- 在同一平面内，过一点有且只有一条直线与已知直线垂直
- 连接直线外一点与直线上各点的所有线段中，垂线段最短
- 直线外一点到这条直线的垂线段的长度，叫做点到直线的距离

### 5.2 平行线及其判定

**平行线的定义**
- 在同一平面内，不相交的两条直线叫做平行线
- 记作：a∥b

**平行公理**
- 经过直线外一点，有且只有一条直线与这条直线平行

**平行公理的推论**
- 如果两条直线都与第三条直线平行，那么这两条直线也互相平行
- 若 a∥b，b∥c，则 a∥c

**平行线的判定**
1. 同位角相等，两直线平行
2. 内错角相等，两直线平行
3. 同旁内角互补，两直线平行

**三线八角**
- 两条直线被第三条直线所截
- 同位角（位置相同）：∠1与∠5，∠2与∠6，∠3与∠7，∠4与∠8
- 内错角（内部交错）：∠3与∠6，∠4与∠5
- 同旁内角（内部同旁）：∠3与∠5，∠4与∠6

### 5.3 平行线的性质

**性质**
1. 两直线平行，同位角相等
2. 两直线平行，内错角相等
3. 两直线平行，同旁内角互补

**命题**
- 判断一件事情的语句
- 结构：题设（已知条件）+ 结论
- 真命题：题设成立时，结论一定成立
- 假命题：题设成立时，结论不一定成立
- 定理：经过推理证明的真命题

**证明**
- 推理的过程叫做证明

### 5.4 平移

**平移的定义**
- 把一个图形整体沿某一直线方向移动
- 这种移动叫做平移变换，简称平移

**平移的性质**
- 平移后得到的新图形与原图形的形状和大小完全相同
- 新图形中的每一点，都是由原图形中的某一点移动后得到的
- 连接各组对应点的线段平行（或在同一直线上）且相等

## 第六章 实数

### 6.1 平方根

**算术平方根**
- 一般地，如果一个正数x的平方等于a，即 x² = a
- 那么这个正数x叫做a的算术平方根
- 记作：√a，读作"根号a"
- 规定：0的算术平方根是0

**平方根**
- 一般地，如果一个数的平方等于a，那么这个数叫做a的平方根或二次方根
- 若 x² = a，则x是a的平方根
- 一个正数有两个平方根，它们互为相反数
- 0的平方根是0
- 负数没有平方根

**开平方**
- 求一个数a的平方根的运算叫做开平方

**重要公式**
- (√a)² = a (a ≥ 0)
- √(a²) = |a|

### 6.2 立方根

**立方根的定义**
- 一般地，如果一个数的立方等于a，那么这个数叫做a的立方根或三次方根
- 若 x³ = a，则x是a的立方根
- 记作：³√a，读作"三次根号a"

**性质**
- 正数的立方根是正数
- 负数的立方根是负数
- 0的立方根是0

**开立方**
- 求一个数的立方根的运算叫做开立方

**重要公式**
- (³√a)³ = a
- ³√(a³) = a

### 6.3 实数

**无理数**
- 无限不循环小数叫做无理数
- 例：√2、√3、π

**实数的定义**
- 有理数和无理数统称实数

**实数的分类**
```
实数
├── 有理数
│   ├── 整数
│   └── 分数
└── 无理数
    ├── 正无理数
    └── 负无理数
```

**实数与数轴上的点**
- 实数与数轴上的点是一一对应的
- 每一个实数都可以用数轴上的一个点来表示
- 反过来，数轴上的每一个点都表示一个实数

**实数的性质**
- 相反数：a的相反数是-a
- 绝对值：|a| = { a, a≥0; -a, a<0 }
- 倒数：a的倒数是1/a (a≠0)

**实数的运算**
- 有理数的运算律在实数范围内仍然适用

## 第七章 平面直角坐标系

### 7.1 平面直角坐标系

**有序数对**
- 有顺序的两个数a与b组成的数对，叫做有序数对
- 记作：(a, b)

**平面直角坐标系**
- 在平面内画两条互相垂直、原点重合的数轴
- 水平的数轴称为x轴或横轴，习惯上取向右为正方向
- 竖直的数轴称为y轴或纵轴，习惯上取向上为正方向
- 两坐标轴的交点为平面直角坐标系的原点

**点的坐标**
- 平面内任意一点P，过P分别向x轴、y轴作垂线
- 垂足在x轴、y轴上对应的数a、b分别叫做点P的横坐标、纵坐标
- 有序数对(a, b)叫做点P的坐标
- 记作：P(a, b)

**象限**
- 坐标平面被两条坐标轴分成I、II、III、IV四个部分
- 每个部分称为象限
- I象限（+，+）、II象限（-，+）
- III象限（-，-）、IV象限（+，-）
- 坐标轴上的点不属于任何象限

**坐标轴上点的特征**
- x轴上的点，纵坐标为0：(a, 0)
- y轴上的点，横坐标为0：(0, b)
- 原点：(0, 0)

### 7.2 坐标方法的简单应用

**用坐标表示地理位置**
- 建立坐标系，选择一个适当的参照点为原点
- 确定x轴、y轴的正方向
- 确定适当的比例尺
- 画出点，写出各点的坐标和地点名称

**用坐标表示平移**
- 点左右平移：横坐标变化，纵坐标不变
  - 向右平移a个单位：(x, y) → (x+a, y)
  - 向左平移a个单位：(x, y) → (x-a, y)
- 点上下平移：纵坐标变化，横坐标不变
  - 向上平移b个单位：(x, y) → (x, y+b)
  - 向下平移b个单位：(x, y) → (x, y-b)

**图形的平移**
- 图形平移后，各点的坐标变化规律相同

## 第八章 二元一次方程组

### 8.1 二元一次方程组

**二元一次方程**
- 含有两个未知数，并且含有未知数的项的次数都是1的方程
- 一般形式：ax + by = c (a≠0, b≠0)

**二元一次方程的解**
- 使二元一次方程两边的值相等的两个未知数的值
- 二元一次方程有无数个解

**二元一次方程组**
- 方程组中有两个未知数，含有每个未知数的项的次数都是1
- 并且一共有两个方程
- 一般形式：
```
{ ax + by = c
{ dx + ey = f
```

**二元一次方程组的解**
- 方程组的两个方程的公共解
- 二元一次方程组一般只有一个解

### 8.2 消元——解二元一次方程组

**代入消元法**
1. 从方程组中选一个系数比较简单的方程，将其中一个未知数用含另一个未知数的代数式表示出来
2. 把所得的式子代入另一个方程中，消去一个未知数，得到一个一元一次方程
3. 解这个一元一次方程，求出一个未知数的值
4. 把求得的未知数的值代入第一步所得的式子中，求出另一个未知数的值
5. 写出方程组的解

**加减消元法**
1. 方程组的两个方程中，如果同一个未知数的系数既不互为相反数也不相等，就用适当的数去乘方程的两边，使同一个未知数的系数互为相反数或相等
2. 把两个方程的两边分别相加或相减，消去一个未知数，得到一个一元一次方程
3. 解这个一元一次方程，求出一个未知数的值
4. 把求得的未知数的值代入原方程组中的任意一个方程，求出另一个未知数的值
5. 写出方程组的解

**选择方法**
- 当方程组中某一个未知数的系数为1或-1时，用代入消元法较简便
- 当方程组中同一个未知数的系数的绝对值相等或成整数倍时，用加减消元法较简便

### 8.3 实际问题与二元一次方程组

**列方程组解应用题步骤**
1. 审题，弄清题意及题目中的数量关系
2. 设未知数（可直接设元，也可间接设元）
3. 找出题目中的相等关系
4. 列出方程组
5. 解方程组
6. 检验并写出答案

**常见类型**
- 和差倍分问题
- 行程问题
- 工程问题
- 销售问题
- 几何问题

### 8.4 三元一次方程组的解法

**三元一次方程组**
- 方程组含有三个未知数，每个方程中含未知数的项的次数都是1
- 并且一共有三个方程

**解法**
- 通过"代入"或"加减"进行消元
- 把"三元"化为"二元"，再化为"一元"

## 第九章 不等式与不等式组

### 9.1 不等式

**不等式的定义**
- 用符号"<"或">"表示大小关系的式子，叫做不等式
- 用符号"≠"表示不等关系的式子也是不等式

**不等式的解**
- 使不等式成立的未知数的值

**不等式的解集**
- 一个含有未知数的不等式的所有解
- 组成这个不等式的解的集合，简称解集

**解不等式**
- 求不等式的解集的过程

**不等式的性质**
- 性质1：不等式两边加（或减）同一个数（或式子），不等号的方向不变
  - 若 a > b，则 a ± c > b ± c
- 性质2：不等式两边乘（或除以）同一个正数，不等号的方向不变
  - 若 a > b，c > 0，则 ac > bc，a/c > b/c
- 性质3：不等式两边乘（或除以）同一个负数，不等号的方向改变
  - 若 a > b，c < 0，则 ac < bc，a/c < b/c

**用数轴表示解集**
- ">" 或 "<"：空心圆圈
- "≥" 或 "≤"：实心圆点
- 大于向右画，小于向左画

### 9.2 一元一次不等式

**一元一次不等式的定义**
- 含有一个未知数，未知数的次数是1的不等式

**解一元一次不等式的步骤**
1. 去分母（注意：两边同乘分母的最小公倍数，不等号方向可能改变）
2. 去括号
3. 移项
4. 合并同类项
5. 系数化为1（注意：两边同除以负数时，不等号方向改变）

**列不等式解应用题**
- 审题，找出不等关系
- 设未知数
- 列不等式
- 解不等式
- 检验并写出答案

### 9.3 一元一次不等式组

**一元一次不等式组**
- 把两个一元一次不等式合起来，组成一个一元一次不等式组

**不等式组的解集**
- 不等式组中几个不等式的解集的公共部分

**解不等式组**
- 求不等式组的解集的过程

**解集的确定方法（同大取大，同小取小）**
- 若 a > b：
  - { x > a, x > b } → x > a（同大取大）
  - { x < a, x < b } → x < b（同小取小）
  - { x > b, x < a } → b < x < a（大小小大中间找）
  - { x > a, x < b } → 无解（大大小小无解了）

### 9.4 课题学习 利用不等关系分析比赛

**应用不等式组解决实际问题**
- 分析问题中的不等关系
- 列出不等式组
- 求解并分析结果

## 第十章 数据的收集、整理与描述

### 10.1 统计调查

**收集数据的常用方法**
- 问卷调查
- 访谈
- 查阅资料
- 实地调查
- 试验

**统计调查的分类**
- 全面调查（普查）：对全体对象进行调查
- 抽样调查：从全体对象中抽取部分对象进行调查

**总体、个体、样本、样本容量**
- 总体：所要考察的全体对象
- 个体：组成总体的每一个考察对象
- 样本：从总体中抽取的一部分个体
- 样本容量：样本中个体的数目

**简单随机抽样**
- 总体中的每一个个体都有相等的机会被抽到

### 10.2 直方图

**频数分布表**
- 把所有数据分成若干组，每个小组的两个端点之间的距离（组内数据的取值范围）称为组距
- 落在各个小组内的数据的个数叫做频数
- 列出频数的分布表，称为频数分布表

**频数分布直方图**
- 用长方形的长表示频数，宽表示组距
- 能够显示各组频数分布的情况

**画频数分布直方图的步骤**
1. 计算最大值与最小值的差（极差）
2. 决定组距与组数
3. 列频数分布表
4. 画频数分布直方图

### 10.3 课题学习 从数据谈节水

**数据处理的一般过程**
1. 收集数据
2. 整理数据（制作统计表）
3. 描述数据（制作统计图）
4. 分析数据
5. 撰写调查报告
6. 交流

**统计图的选择**
- 条形统计图：显示具体数据，易于比较
- 折线统计图：显示数据变化趋势
- 扇形统计图：显示各部分占总体的百分比
- 直方图：显示数据分布情况
